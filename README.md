# NCAAFB-Data
Sports Analytics on NCAAFB Data Using Real-Time API Feeds
**********************************************************
Sports Analytics on NCAAFB Data Using Real-Time API Feeds
**********************************************************

###Description###
	This project demonstrates how to integrate Sportradar APIs with Streamlit for interactive dashboards.
	It connects to a specific endpoint of the Sportradar API for Football to retrieve real-time data and visualizes it using the Streamlit framework. It serves as a basic template for building interactive data dashboards with live sports information.

**********************************************************

###Pre-Requisites####
  Latest google chrome or Microsoft Edge Browser
  Python (3.8 or later recommended)
  Streamlit
  A Sportradar API Key: You can obtain one by registering on the   Sportradar Developer Portal.
  SQL database (MySQL Workbench)

**********************************************************

Project Structure

streamlit-sportradar-demo/
